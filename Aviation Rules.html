Aviation Rules
FRANCE — SERA + compléments ›
1) Les réflexes “toujours” (peu importe l’espace aérien)
  • See & Avoid : en VFR tu restes responsable de l’anticollision, même si tu as une clairance (sauf séparation ATC spécifique).
  • Priorités (logique ICAO/SERA) :
  • Convergence : celui qui voit l’autre sur sa droite cède.
  • Face à face : chacun évite par la droite.
  • Dépassement : tu dépasses en restant bien clair (généralement par la droite en pratique).
  • Atterrissage : un aéronef en finale/atterrissage est prioritaire (sans “forcer le passage”).
  • Anti-collision au sol/vol, éclairage nuit, vigilance intégration : habitudes pro (pas “réglementaire partout”, mais discipline pilote pro).
2) Hauteurs minimales (le “plancher” à ne pas violer)
  • Socle : SERA + restrictions nationales possibles (survol d’agglomérations, rassemblements, bruit, zones sensibles).
  • Principe : ne pas voler si bas que tu ne peux pas gérer une panne “sans danger indu” + marges minimales selon zone/obstacles (références SERA).
  • France : attention aux textes nationaux (ex. arrêtés sur survol agglos/rassemblements) ; l’AIP rappelle que les minima SERA s’appliquent sans préjudice de ces règles françaises.
  • Habitude pilote : avant vol, repérer agglos / rassemblements / sites sensibles + décider d’une hauteur plancher perso (souvent > minima).
3) VMC / météo VFR : les minima “visibilité + distance aux nuages”
  • Les vols VFR doivent respecter les VMC minima (table SERA) sauf Special VFR (SVFR).
  • Habitude pilote : “si je dois argumenter que je suis VMC, je suis déjà trop près”.
4) Altitudes “semi-circulaires” (niveaux de croisière)
  • En Europe, la règle de base : niveaux de croisière en altitudes sous la TA (transition altitude) et en flight levels au-dessus.
  • En pratique VFR : on applique la règle semi-circulaire (niveaux selon la route magnétique) quand on n’est pas sous guidage ATC, typiquement au-dessus d’un seuil publié/commun (souvent 3000 ft sol/eau dans beaucoup de publications) ; l’AIP France la traite dans ses sections règles VFR.
  • Habitude pilote : annoncer/penser “cap → niveau” (évite de te retrouver à contre-parité).
5) Espaces aériens (France) : ce que ça “veut dire” en pratique pilote
  • France : classes ICAO A à G + réalité terrain :
  • Classe A : IFR uniquement (en pratique).
  • Classe C / D : contrôlé ; entrée VFR = généralement contact radio + clairance/autorisation (selon classe/secteur).
  • Classe E : contrôlé pour IFR, VFR souvent sans clairance (mais règles VMC + vigilance).
  • Classe G : non contrôlé (auto-information, vigilance, parfois fréquences).
  • Couches/règles fréquentes :
  • CTR / TMA : zones autour des terrains contrôlés (souvent classe D/E selon endroits).
  • Zones réglementées : P / R / D : vérifier conditions d’activité (AIP/NOTAM) avant traversée.
  • RMZ/TMZ : radio/transpondeur obligatoires si publié.
  • Habitude pilote : lecture carte = “classe + plancher/plafond + exigences (radio, transpondeur) + activités (R/D)”.
6) Radio / clairances : les automatismes
  • Quand tu dois parler : dès que l’espace l’exige (CTR/TMA contrôlée, RMZ, zones avec service ATS publié) + quand l’intégration l’impose (sécurité).
  • Clairance ≠ autorisation de tout : une clairance de transit ne t’exonère pas des minima VMC, des hauteurs mini, ni de la vigilance trafic.
7) Plan de vol, transpondeur, équipements : l’esprit
  • Plan de vol : requis dans certains cas (traversée frontière, IFR, certains vols spécifiques, etc.) — décrit par l’AIP (sections “Flight planning” + règles VFR).
  • Équipement : dépend du type de vol (jour/nuit, radio requise, transpondeur requis, éventuellement GNSS/8.33, etc.) → c’est AIP + réglementation avion qui commandent.
8) Nuit VFR (France) : possible, mais encadré
  • La France autorise le VFR de nuit sous conditions, avec des niveaux minimaux et des règles spécifiques décrites dans l’AIP (ENR 1.2).
  • Habitude pilote : nuit = marges +++ (météo, carburant, terrains de dégagement, altitudes).
USA — FAA (Part 91 + AIM) ›
1) Minimum safe altitudes (le classique à connaître par cœur)
  • 14 CFR 91.119 :
  • Congested area / rassemblement : 1000 ft au-dessus de l’obstacle le plus haut dans un rayon horizontal de 2000 ft.
  • Autres zones : 500 ft AGL, sauf au-dessus d’eau/zone peu peuplée : pas plus près que 500 ft de toute personne/véhicule/structure.
  • Habitude pilote US : “je suis congested ou pas ?” (ça change tout) + attention aux rassemblements (stades, événements).
2) VFR weather minimums (visibilité + clairance nuages)
  • 14 CFR 91.155 fixe les minima selon classe d’espace et altitude.
  • Habitude pilote : savoir instantanément la règle “3-152” (3 SM / 500 below / 1000 above / 2000 horizontal) en Class E sous 10 000 ft, et les exceptions Class G, Class B “clear of clouds”, etc. (l’AIM résume très bien).
3) VFR cruising altitudes (semi-circulaire version US)
  • 14 CFR 91.159 (en croisière au-dessus de 3000 ft AGL, hors exceptions) :
  • Course magnétique 000–179° : odd thousands + 500 (ex : 3500, 5500, 7500).
  • 180–359° : even thousands + 500 (ex : 4500, 6500, 8500).
4) Airspace US : ce que ça implique (très “procédural”)
  • Classes : A/B/C/D/E/G, exigences d’entrée codifiées :
  • Class B : clairance explicite (“Cleared into the Bravo”).
  • Class C/D : contact radio + instructions (et souvent transpondeur).
  • Class E : contrôlé pour IFR, VFR souvent sans clairance sauf surface area/conditions spéciales.
  • Class G : non contrôlé (mais règles VFR + vigilance).
  • L’AIM donne la vue d’ensemble + les minima VFR par classe.
5) Speed limits (souvent oubliés)
  • 14 CFR 91.117 :
  • ≤ 250 KIAS sous 10 000 ft MSL (sauf autorisation).
  • ≤ 200 KIAS (cas fréquents) : sous certaines zones (ex. sous certains planchers Bravo, proche aéroports C/D à basse hauteur – cf. texte complet).
6) Right-of-way rules (à connaître)
  • 14 CFR 91.113 : règles de priorité (convergence, face à face, dépassement, atterrissage).
7) Carburant VFR (super concret, super contrôlé)
  • 14 CFR 91.151 :
  • VFR jour : assez pour aller au point prévu + 30 min.
  • VFR nuit : + 45 min.
  • (Rotorcraft : règle spécifique).
8) Équipement minimum (VFR day/night, IFR)
  • 14 CFR 91.205 : liste des instruments/équipements requis selon VFR jour, VFR nuit, IFR.
  • Habitude pilote : si un équipement requis est inop, tu appliques la logique MEL/KOEL/91.213 (pas “on verra bien”).
9) Transpondeur / ADS-B Out (réalité US moderne)
  • ADS-B Out requis dans la plupart des espaces contrôlés définis par la règle (Class A, B, C, et d’autres volumes autour/au-dessus, etc.).
  • Cadre réglementaire ADS-B : 91.225.
  • Habitude pilote : “où suis-je par rapport à la rule airspace ADS-B / Mode C veil ?” avant même de démarrer.
Mini check (valable France & US) ›
  • 1) Où je vole ? (classe, plancher/plafond, zones P/R/D, RMZ/TMZ / ADS-B/Mode C veil).
  • 2) Suis-je légal météo ? (VMC minima / 91.155 / SERA).
  • 3) Suis-je légal hauteur ? (minima + congested/agglos + rassemblements).
  • 4) Mon niveau de croisière est cohérent ? (semi-circulaire + TA/FL en Europe).
  • 5) Radio & clairances : ai-je le bon contact/autorisation ?
  • 6) Carburant & dégagements : réserve légale + marge perso.
  • 7) Équipement : requis OK (jour/nuit/IFR, transpondeur/ADS-B, lumières).
  • 8) Priorités & intégration : see&avoid, annonces, discipline.
